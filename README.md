# 📐 DDD ModelStudio v1.0.0

**The High-Fidelity Strategic & Tactical Modeler for Modern Architects.**

DDD ModelStudio is a professional-grade, interactive web application designed to bridge the gap between high-level **Strategic Domain Mapping** and low-level **Tactical Component Design**. It combines the principles of **Domain-Driven Design (DDD)** with the power of **Enterprise Integration Patterns (EIP)** in a seamless, zero-build environment.

---

## 🎯 The Vision: Why DDD ModelStudio?

Modern systems are complex. Traditional diagramming tools are either too generic (boxes and lines) or too rigid (UML). DDD ModelStudio was created to provide a **semantic modeling environment** where the tools understand the architecture.

- **Ubiquitous Language:** Directly model Aggregates, Entities, and Bounded Contexts using the language of your domain.
- **Resilient Patterns:** Natively design fault-tolerant systems using built-in Resilience patterns like Circuit Breakers and Retries.
- **Fluid Orchestration:** Visualize complex messaging flows with the full catalog of 65+ Enterprise Integration Patterns.
- **Architectural Integrity:** Our unique "Push-to-Expand" spatial engine ensures that your strategic boundaries are respected and maintained as your model grows.

## 🚀 Key Features Manifest

### 🏛️ Strategic Design & Spatial Physics
- **Intelligent Boundaries:** Native support for Bounded Contexts, Core/Supporting/Generic Subdomains, and Big Ball of Mud.
- **Dynamic Containment:** Boundaries automatically expand to fit their children. No more manual resizing of boxes.
- **Recursive Pushing:** Moving or expanding a container gently pushes neighbors away, keeping your entire map organized and collision-free.
- **Eject Actions:** Instantly move elements out of a context with the "Exit Context" command.

### 🔗 The EIP Engine (65+ Patterns)
- **Comprehensive Catalog:** Full support for Messaging Channels, Routing (Choice, Splitter, Aggregator), Transformation (Enricher, Normalizer), and System Management (Wire Tap, Control Bus).
- **Resilience Palette:** Dedicated tools for **Circuit Breakers**, **Retry Policies**, **Timeouts**, and **Dead Letter Queues (DLQ)**.
- **Animated Flows:** Connections default to an animated, dashed style to visualize the living heartbeat of your data.
- **Advanced Connections:** Full control over directionality (one-way, reverse, both), line patterns, and midpoint labeling.

### 📝 Technical Element Inspector
- **Structured Fields:** Define data fields with strict typing (String, Int, Float, Date, Time, DateTime) and auto-generated sample data.
- **Specialized Editors:**
  - **Tables:** Professional inline row/column editor for relational design.
  - **Messages:** Dedicated Header and Payload blocks for precise event modeling.
  - **Collections:** Large-scale JSON editor with real-time validation and formatting.

### 🎨 Visuals & Enterprise Branding
- **Custom Image Logos:** Paste standard image URLs (PNG/SVG) into any element to render official tech brands (Oracle, MongoDB, Kafka, etc.) or corporate bank logos.
- **Categorized Icon Picker:** Searchable modal with icons grouped by Architecture, Security, Data, and Tech Brands.
- **Multi-Theme Engine:**
  - **Pro Dark:** Deep aesthetics for long design sessions.
  - **Paper Light:** High-contrast enterprise look for documentation.
  - **VSCode (Dark/Light):** Familiar editor-style aesthetics.
  - **Enterprise Zinc & Cyberpunk:** Minimalist and high-contrast alternatives.

### 💾 Engineering Workflow
- **Save & Load (.jc):** Full workspace serialization into portable JSON blueprints.
- **SVG Export:** Generate high-fidelity vector images for documentation, presentations, or GitHub READMEs.
- **Interactive Help:** A comprehensive, searchable documentation portal built directly into the app.
- **Audio Feedback:** A synthesized Web Audio engine providing tactile confirmation for every action.

## 🛠️ Installation & Usage

DDD ModelStudio is a **Vanilla Web App**. No build steps. No `node_modules`. No dependencies.

1. **Clone the Repo:** `git clone https://github.com/geekpratyush/DDDModelStudio.git`
2. **Launch:** Double-click `index.html` to open it in any modern browser.
