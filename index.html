Keyboard Shortcuts

  • V: Move Objects mode.
  • H: Pan Canvas mode (Middle click also pans).
  • M: Marquee Select mode.
  • 0: Reset Zoom/Pan.
  • +/-: Zoom In/Out.
  • Ctrl/Cmd+A: Select All.

Containment Physics

Bounded Contexts automatically expand to fit their children. Dragging elements against the inner boundary pushes it outwards. Containers also push each other to prevent overlapping.

Advanced Connections

Right-click any connection to change color, line pattern (solid, dashed, dotted), or toggle the animated flow. You can also set one-way, reverse, or bidirectional arrows.

Tactical Editors

Specialized editors are available for different node types. Tables support inline row editing, Messages have dedicated header/payload blocks, and Collections include a JSON formatter.

Enterprise Integration Patterns (EIP)

The studio supports the full catalog of 65+ standard patterns for modeling robust distributed systems:

  • Resilience: Circuit Breaker, Retry, Timeout, and Dead Letter Queue (DLQ). These are vital for fault-tolerant messaging.
  • Routing: Choice (Content-Based), Splitter, Aggregator, Dynamic Router, and Recipient List.
  • Transformation: Content Enricher, Normalizer, and Claim Check (for large payloads).

Explore the Templates menu to see these in action, including complex Scatter-Gather and Resilience Flow blueprints.

Branding & Logos

Paste any image URL (PNG/SVG) into the Icon field of an element to render a custom logo. Native support for major tech brands like Oracle, Kafka, and MongoDB is included in the picker.

Save & Export

Save your work as a .jc (JSON) file to keep all nodes and custom templates. Export as SVG for high-quality architectural documentation.
